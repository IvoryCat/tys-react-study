## 리액트 프로젝트 생성

npx create-react-app [folder]

## 기존 프로젝트에 리액트 주입

리액트 script 코드 삽입 + 바벨 코드 삽입

## useState

1. 값이 바뀌면서
2. 화면에 출력될 때 사용

useState 함수는 Hook이고, 함수형 형식에서만 사용.

## Programming paradigm

- 명령형 : 동작을 지시
- 선언형 프로그래밍 : 상태를 서술
- 함수형 프로그래밍
  - map
  - filter
  - forEach
  - reduce

## Design 
- 응집도: 하나의 목적을 가진 코드를 뭉치자.
- 단일책임: 함수가 한가지 기능을 하도록 하자
- 추상화: 함수의 세부구현 단계를 일치시키자
